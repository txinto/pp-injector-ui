# Provisioning

Component to provision Wi-Fi credentials at runtime using ESP-IDF's provisioning
framework and a Bluetooth mobile app.

Do not enable both `Provisioning` and `Wifi` in the same project variant.

## Documentation
Details for Wi-Fi credential provisioning are here:
https://docs.espressif.com/projects/esp-idf/en/stable/esp32/api-reference/provisioning/wifi_provisioning.html

That page also links to the Android and iOS provisioning apps.
